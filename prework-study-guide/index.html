HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The body is the parent element and elements that are nested inside are called child element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A margin acts much like a margin in a word processing doc
  • It is not visible on the page, merely adds a buffer
  • When set margin property to 0, we are indicating that we want NO extra space around that element.
  • A padding indiscated how much we want around the content inside an element.
  • Padding property adds space around the content inside an element.
  • When you increase padding, you are actually marking the entire element BIGGER!
  • That's because, padding, while transparent, is located inside the element, whereas the margin is the transparent buffer outside the element.

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git commit: commands with a commit message that describes what the commits contains
  • git pull origin main: before we push this up to our repository, we double check our local branch is sync with the base branch in Github
  • git push: push the changes we have locally in our remote Github branch
